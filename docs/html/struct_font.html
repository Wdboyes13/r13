Detailed Description

A struct containing data for a font This font struct is to use and load fonts the default font is the VT323 font and is usually loaded automatically when you instantiate the R13 class the default font is selected when the "path" argument of the constructor is "__DEFAULT__".

Example usage:

Font my_font("path/to/my_font.ttf")
r13->set_font(my_font)
